# Smart Weather Monitoring System

This project monitors temperature, humidity, air quality, and rainfall using Arduino UNO R4 WiFi and IoT.

## Features
- Real-time weather monitoring
- ThingSpeak cloud integration
- Web-based dashboard
- Mobile-friendly interface

## Technologies Used
- Arduino UNO R4 WiFi
- DHT11 Sensor
- MQ-2 Gas Sensor
- Rain Sensor
- ThingSpeak
- HTML, CSS, JavaScript

## How to Use
1. Upload Arduino code
2. Connect sensors
3. Open dashboard
